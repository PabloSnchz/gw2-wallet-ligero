
# MetaEventos — Paquete de imágenes (colocar en /assets/meta/)

Este paquete contiene **placeholders** con los nombres correctos que espera el código:

- assets/meta/choya.jpg — Choya Piñata (Amnoon)
- assets/meta/chak_gerent.jpg — Regente Chak (Tangled Depths)
- assets/meta/vinewrath.jpg — Furienhiedra (Mordrem Vinewrath)
- assets/meta/shatterer.jpg — El Despedazador (The Shatterer)
- assets/meta/dragonstorm.jpg — Tormenta Dragón (Dragonstorm)
- assets/meta/tarir.jpg — Tarir (Auric Basin — Octovino)

## Cómo reemplazarlas
1) Sustituí **cada placeholder** por una imagen **1920×1080** difuminada suave (blur/opacity) con el boss/mapa correspondiente.
2) Conservá **exactamente** los **nombres de archivo**.
3) Subí la carpeta `assets/meta/` completa al repo (GitHub → Add file → Upload files).

> Sugerencias de fuentes: páginas de la **Wiki oficial** (licencia CC BY‑SA) de cada evento/boss. Ajustá brillo/contraste y desenfoque para mantener la legibilidad del texto de la tarjeta.

## Notas
- Si no reemplazás estas imágenes, las tarjetas cargarán el placeholder.
- Los íconos de infusiones **no** van en esta carpeta: el código los descarga del **GW2 API (/v2/items)** según IDs.
