  1. 📱 Entrá a la página: https://pablosnchz.github.io/gw2-wallet-ligero/

  3. 🧭 Explorá los módulos desde la barra lateral

🧙 Asistente de cuentas

Creá tu archivo seguro en 4 pasos:

1
📎 Descargar plantilla
Archivo Excel con columnas predefinidas (id, nombre, email, password, apiKey, etc.)
2
📤 Subir Excel → Generar JSON
Convertí tu Excel a JSON y descargá el archivo
3
🌐 Enriquecer con API (opcional)
Completá datos de GW2 automáticamente (nombre de cuenta, AP, expansiones)
4
🔐 Cifrar con contraseña
Creá tu archivo .enc para cargar en el panel

🔒 Seguridad: Todo ocurre en tu navegador. Ningún dato sale de tu computadora.

💱 Conversor Gem ↔ Gold

  • 💎 Conseguir gemas: Ingresá gemas → te mostramos el costo en oro
  • 🪙 Conseguir oro: Ingresá oro → te mostramos cuántas gemas podés comprar
  • 🎯 Atajos rápidos: 100, 400, 800, 1200 gemas / 10, 100, 250 de oro
  • 📊 Índice de conveniencia: Te dice si es buen momento para comprar (basado en 400 gemas)

🔑 Detección automática de llave semanal

La Bóveda detecta automáticamente si ya reclamaste la llave del León Negro esta semana:

  • ✅ Busca personajes Thief con nivel ≥10
  • ✅ Antigüedad menos de 7 días
  • ✅ Creados después del último reset (lunes 07:30 UTC)

Si creaste un Thief el domingo y al día siguiente es lunes, la llave estará disponible (no bloqueada por el personaje anterior).
