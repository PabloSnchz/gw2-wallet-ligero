🔑 Detección automática de llave semanal

La Bóveda detecta automáticamente si ya reclamaste la llave del León Negro esta semana:

  • ✅ Busca personajes Thief con nivel ≥10
  • ✅ Antigüedad menos de 7 días
  • ✅ Creados después del último reset (lunes 07:30 UTC)

Si creaste un Thief el domingo y al día siguiente es lunes, la llave estará disponible (no bloqueada por el personaje anterior).

💱 Conversor Gem ↔ Gold

  • 💎 Conseguir gemas: Ingresá gemas → te mostramos el costo en oro
  • 🪙 Conseguir oro: Ingresá oro → te mostramos cuántas gemas podés comprar
  • 🎯 Atajos rápidos: 100, 400, 800, 1200 gemas / 10, 100, 250 de oro
  • 📊 Índice de conveniencia: Te dice si es buen momento para comprar (basado en 400 gemas)

💾 Backup y Restaurar (NUEVO)

Guardá toda tu configuración en un archivo para restaurarla cuando quieras o en otro navegador.

Backup

Exportá toda tu configuración a un archivo JSON:
• API Keys
• Wizard's Vault (pins y marcas)
• Wallet (divisas fijadas, snapshots)
• Activities (toggles, home nodes marcados)
• Characters (POIs asignados, ubicaciones)
• Meta (favoritos, "Hecho hoy")
• Configuración global

Cómo usar: Hacé clic en el botón Backup en la barra superior → se descargará un archivo gw2-backup-YYYY-MM-DD.json

Restaurar

Importá un archivo de backup para recuperar toda tu configuración.

Cómo usar: Hacé clic en el botón Restaurar → seleccioná tu archivo JSON → confirmá la sobrescritura → la página se recargará con tu configuración restaurada.

☁️ Sincronización con GitHub Gist (NUEVO)

¿Usás la Bóveda en varios dispositivos? Sincronizá tu configuración en la nube con GitHub Gist.

1
🔑 Generá un token en GitHub
Andá a GitHub Settings → Tokens, hacé clic en "Generate new token (classic)", seleccioná el permiso gist y generá el token.
2
☁️ Configurá la sincronización
En la Bóveda, hacé clic en el botón Sincronizar (icono de nube) → pegá tu token y hacé clic en "Guardar token". La app creará automáticamente un Gist privado.
3
📤 Subí tu configuración
Una vez configurado, podés subir tu configuración actual con el botón "Subir configuración a la nube".
4
📥 Sincronizá en otro dispositivo
En tu otro dispositivo, repetí los pasos 1 y 2 con el mismo token, luego hacé clic en "Sincronizar desde la nube". ¡Toda tu configuración estará igual!

🔒 Seguridad: El token se guarda cifrado en tu navegador. El Gist es privado, solo vos podés acceder con tu token. La app no almacena ninguna información personal en servidores externos.
