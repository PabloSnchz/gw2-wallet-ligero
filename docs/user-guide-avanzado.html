1. Cartera (Wallet)

¿Qué es?

El panel principal que muestra todas tus divisas: oro, gemas, karma, laureles, contratos comerciales, mosaicos de elegía, y cualquier otra moneda del juego.

¿Cómo se usa?

  1. Seleccioná tu API Key en el selector superior
  2. Las divisas se muestran como tarjetas con ícono, nombre y cantidad
  3. Usá los filtros para buscar por nombre, categoría, o ver solo las principales
  4. Alterná entre vista tarjetas y vista tabla con el botón correspondiente
  5. Hacé clic en el 📌 para fijar una divisa arriba de todo

¿Para qué sirve?

  • Ver de un vistazo cuánto oro, gemas y karma tenés
  • Comparar divisas entre cuentas (con el Dashboard multi-cuenta)
  • Fijar las divisas que más te interesan para acceso rápido

2. Meta & Eventos

¿Qué es?

Un panel que muestra todos los meta eventos y world bosses con sus horarios en tu zona horaria local, estado (activo/próximo/más tarde), y si ya los completaste hoy. Las tarjetas tienen una estructura unificada estilo Raids.

¿Cómo se usa?

  1. Las tarjetas muestran nombre, mapa, tipo, y horarios
  2. 🟢 Activo = está ocurriendo ahora. 🟡 Próximo = empieza en ≤20 min. 🔵 Más tarde = después.
  3. Clic en 📌 para fijar hasta 6 favoritos
  4. Clic en Horarios para ver todas las ventanas del día
  5. El botón Refrescar estado consulta si ya lo hiciste hoy
  6. Cada tarjeta tiene una barra de progreso dentro del header
  7. Los horarios se muestran en tu hora local con colores semánticos
  8. Clic en Wiki abre la página en español (wiki-es.guildwars2.com)

¿Para qué sirve?

  • Saber qué world boss o meta evento está por empezar
  • Planificar tu sesión de juego alrededor de los eventos que te interesan
  • Ver de un vistazo qué eventos ya completaste hoy

3. Logros

¿Qué es?

Un panel que muestra tus logros más cercanos a completar, ordenados por porcentaje de progreso, con filtros por categoría, rareza y tipo de recompensa.

¿Cómo se usa?

  1. Usá los dropdowns de filtro para elegir umbral, categoría, y tipo de recompensa
  2. Cada tarjeta muestra nombre del logro, categoría, progreso, y recompensa
  3. Clic en Wiki para abrir la página oficial del logro

¿Para qué sirve?

  • Identificar logros que están a punto de completarse
  • Filtrar por tipo de recompensa (items, títulos, maestrías)
  • Ver el AP total y el AP potencial que podés ganar

4. Cámara del Brujo

¿Qué es?

El módulo del Wizard's Vault con cuatro secciones: objetivos diarios, semanales, especiales, y la tienda de Aclamación Astral.

¿Cómo se usa?

Objetivos

  1. Seleccioná la pestaña correspondiente (Diarias/Semanales/Especiales)
  2. Cada tarjeta muestra objetivo, progreso, recompensa en AA, y estado
  3. Los objetivos tienen un modo (PvE, PvP, WvW) indicado con un ícono

Tienda

  1. Seleccioná la pestaña Tienda
  2. Usá filtros para buscar items por nombre, ordenar por costo, o filtrar por tipo
  3. Cada tarjeta muestra el item, costo en AA, y cuántas veces lo compraste
  4. Clic en 📌 para fijar items que querés comprar
  5. Usá los inputs manuales para marcar compras hechas fuera de la API

¿Para qué sirve?

  • Hacer el seguimiento diario/semanal de objetivos del Wizard's Vault
  • Planificar tus compras de AA (Aclamación Astral)

5. Actividades

¿Qué es?

Un panel que agrupa actividades diarias y semanales: PSNA, refinamiento de Ecto, fractales, llave del León Negro, Piedras de Invocación Vetustas (Leivas), y Home Nodes.

¿Cómo se usa?

PSNA

  • Muestra los 6 agentes del día con sus waypoints. Clic para copiar al portapapeles.

Ecto

  • Los íconos tienen glow: ✅ verde si está hecho, ⏳ ámbar si pendiente.

Fractales

  • Muestra los fractales T4 y recomendados del día.

Llave del León Negro

  • Detecta automáticamente si ya reclamaste la llave esta semana (Thief nivel 10+, <7 días, misma semana).

Leivas

  • Contador manual de 0 a 5 con botones + y −.

Home Nodes

  • 74 nodos de heredad con filtros por categoría, tipo y estado.

¿Para qué sirve?

  • Centralizar todas las tareas diarias/semanales en un solo panel
  • No olvidarte de la llave semanal ni de las Leivas
  • Hacer la ruta de PSNA sin buscar waypoints a mano

6. Inventario y Personajes

¿Qué es?

Un buscador unificado que indexa tu banco, almacenamiento de materiales y armería legendaria. Es la pantalla principal de #/account/characters.

¿Cómo se usa?

  1. Navegá a Inventario y Personajes desde la sidebar
  2. Los KPIs muestran cuántos items tenés en cada sección
  3. Usá el buscador para encontrar objetos por nombre
  4. Filtrá por rareza con el dropdown o los chips
  5. Clic en un KPI o encabezado para ir a la vista detallada
  6. En la vista de Banco, los slots se muestran en grid de 10×3 con paginación cada 30
  7. En la vista de Materiales, los items están organizados en 10 categorías como en el juego
  8. Clic en cualquier ítem abre un modal con stats completos, valor NPC y enlace a Wiki en español

¿Para qué sirve?

  • Encontrar cualquier objeto sin tener que abrir el juego
  • Saber cuánto espacio ocupás en el banco
  • Ver tus legendarios en la armería

7. Dashboard de Inventario Multi-Cuenta NUEVO

¿Qué es?

Un dashboard que muestra los ítems del inventario (banco + materiales + personaje activo) de todas tus cuentas en una tabla comparativa. Ruta: #/inventory/dashboard.

¿Cómo se usa?

  1. Navegá a Inventario y Personajes → botón Dashboard en el toolbar
  2. Seleccioná uno de los 3 sets predefinidos: Alto Valor, Materiales de artesanía, o Símbolos y demás
  3. En el set "Materiales de artesanía", podés activar/desactivar tiers (T6, T5, T4, T3) con checkboxes
  4. Usá el dropdown de ítems para seleccionar qué ítems querés ver en la tabla
  5. La tabla muestra cada cuenta como fila y cada ítem como columna
  6. Los KPIs superiores muestran cantidades totales y valor en oro usando precios del Trading Post
  7. Ordená por cualquier columna haciendo clic en el encabezado
  8. Usá los filtros para ocultar cuentas vacías, columnas vacías o cuentas main
  9. Cuando el personaje activo aporta ítems nuevos, la celda parpadea en ámbar y queda fija hasta que pasás el mouse

¿Para qué sirve?

  • Comparar el inventario entre todas tus cuentas
  • Saber qué cuenta tiene más materiales de artesanía de cierto tier
  • Calcular el valor total de tu inventario en oro
  • Identificar cuentas con materiales valiosos acumulados

8. Conversor de Divisas

¿Qué es?

Una herramienta para calcular el costo de gemas en oro y viceversa, con tabs adicionales para transacciones activas e items populares del TP. Se accede desde un modal.

¿Cómo se usa?

Tab Cambio

  1. Abrí el conversor desde Cartera → botón 💎 Conversor
  2. Ingresá gemas u oro, o usá los atajos rápidos (100, 400, 800, 1200 gemas / 10g, 100g, 250g)
  3. El índice de conveniencia te dice si es buen momento para comprar (basado en 400 gemas)

Tab Transacciones

  1. Requiere API Key con permiso tradingpost
  2. Muestra tus órdenes activas de compra/venta con KPIs de totales (compras, ventas, balance)

Tab Populares

  1. Items con mayor volumen en el TP, filtrables por rareza
  2. Ideal para ver qué legendarias se venden más rápido

¿Para qué sirve?

  • Saber cuánto oro necesitás para comprar X gemas
  • Evaluar si conviene comprar gemas ahora o esperar
  • Monitorear tus órdenes activas en el TP sin abrir el juego

¿Cómo se usa?

  1. Navegá a Raids desde la sidebar
  2. En el header, hacé clic en el botón Strikes
  3. Se muestra el grid de strikes organizado por expansión
  4. Cada strike tiene su estado (✅ Completada / ❌ Pendiente) y badge de dificultad
  5. Clic en Detalle para ver descripción, estrategia y enlaces a wiki/video
  6. Usá los botones Raids y Strikes para volver a Raids o mantenerte en Strikes
  7. La vista activa (Raids o Strikes) se guarda automáticamente en localStorage

¿Para qué sirve?

  • Llevar el control semanal de todas las Strike Missions
  • Ver qué strikes completaste y cuáles te faltan
  • Acceder rápido a guías y videos de cada strike
  • Calcular cuántos LI podés farmear adicionalmente (15 por semana)

¿Qué es?

¿Cómo se usa?

  1. Andá a Cartera → botón 📊 Dashboard
  2. Seleccioná divisas y ordená por columna

12. Panel de Cuentas

¿Qué es?

Un gestor seguro de múltiples cuentas de GW2 con cifrado local. Diseño "Profile Card" premium con jerarquía visual de 3 zonas.

¿Cómo se usa?

  1. Usá el Asistente (4 pasos) para crear tu archivo .enc desde Excel
  2. O cargá directamente un archivo .enc con tu contraseña
  3. Alterná entre vista tarjetas (Profile Card) y vista tabla con zebra striping
  4. Las expansiones son colapsables con barra de progreso
  5. Twitch y GeForce Now están siempre visibles con íconos de estado
  6. Clic en cualquier campo para copiarlo al portapapeles

¿Para qué sirve?

  • Gestionar múltiples cuentas sin tener que recordar contraseñas
  • Tener a mano los datos de Twitch y GeForce Now de cada cuenta
  • Ver qué expansiones tiene cada cuenta

13. Purchase Detail

¿Qué es?

Un panel de seguimiento de compras del Wizard's Vault que muestra tus cuentas, items fijados, y progreso.

¿Cómo se usa?

  1. Accedé desde la Tienda WV → botón Compras en el nav de tabs
  2. La tabla muestra cada cuenta como fila y cada item fijado como columna
  3. KPIs superiores: AA disponible, necesaria, y delta
  4. Usá los inputs manuales para marcar compras hechas
  5. Cada celda tiene una barra de progreso compacta
  6. El estado online se detecta automáticamente (actividad en los últimos 20 minutos)

¿Para qué sirve?

  • Ver de un vistazo qué items faltan comprar en cada cuenta
  • Planificar cuánto AA necesitás farmear
  • Identificar cuentas con déficit de AA

14. Dashboard de Objetivos WV

¿Qué es?

Un panel que compara los objetivos semanales del Wizard's Vault de todas tus cuentas en una sola tabla. Cada celda muestra el estado de un objetivo para una cuenta.

¿Cómo se usa?

  1. Navegá a Cámara del Brujo → pestaña Semanales (o Diarias/Especiales)
  2. Clic en el botón Dashboard (margen derecho de la fila de tabs)
  3. La tabla muestra filas (cuentas) y columnas (objetivos) con ✅ Reclamado, ✔️ Completado, o progreso
  4. Los KPIs superiores muestran Cuentas, Reclamados, Completados, y Progreso con mini barra
  5. El countdown semanal muestra cuánto falta para el reset del lunes
  6. La fila TOTAL resume cuántas cuentas completaron cada objetivo
  7. Usá Refrescar y Volver para navegar

¿Para qué sirve?

  • Ver de un vistazo qué cuentas completaron los semanales
  • Identificar cuentas que no reclamaron sus recompensas
  • Planificar en qué cuentas farmear AA antes del reset
  • Comparar progreso entre cuentas sin cambiar de API Key

15. Backup y Restaurar

¿Qué es?

Un sistema para exportar/importar toda tu configuración entre navegadores.

¿Cómo se usa?

  1. Clic en Backup → descarga un archivo JSON
  2. En otro navegador, clic en Restaurar → seleccioná el archivo

¿Qué datos se guardan?

  • API Keys (nombres, keys, key seleccionada)
  • Wizard's Vault (pins y marcas por temporada)
  • Wallet (divisas fijadas, snapshots, vista compacta)
  • Activities (toggles, home nodes marcados)
  • Characters (POIs asignados, historial de ubicaciones)
  • Meta (favoritos, "Hecho hoy")
  • Configuración global (welcome visto)

16. Sincronización con GitHub Gist

¿Qué es?

Sincronización opcional de configuración en la nube con GitHub Gist.

¿Cómo se usa?

  1. Generá un token de GitHub con permiso gist en GitHub Settings
  2. En la Bóveda: clic en Sincronizar (icono de nube) → pegar token → Guardar
  3. Usá Subir configuración a la nube para guardar tus datos en el Gist
  4. En otro dispositivo, repetí los pasos 1 y 2, luego usá Sincronizar desde la nube

Seguridad

  • El token se guarda cifrado en tu navegador
  • El Gist es privado, solo vos podés acceder con tu token
  • La app no almacena ninguna información personal en servidores externos

17. API Keys

¿Qué son?

Claves de acceso para consultar tus datos de GW2. Se gestionan desde el modal que se abre con el botón "Gestión de API Keys" en el header.

Gestión de API Keys

  • Agregar: pegá tu API Key, asignale un nombre opcional y guardá
  • Usar: clic en "Usar" en la key que quieras activar
  • Renombrar: clic en "Renombrar" para cambiar el nombre visible
  • Copiar: clic en "Copiar" para copiar la key al portapapeles
  • Eliminar: clic en "Eliminar" para borrar la key de tu navegador
  • Las keys se guardan localmente en tu navegador, nunca se envían a ningún servidor
