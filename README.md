# GW2 Wallet — v1.6.0-alpha1

Esta versión **alfa** agrega la nueva pantalla **MetaEventos** SIN modificar el diseño general ni romper la funcionalidad actual de Wallet.

- Navegación: nuevo tab **MetaEventos**.
- Grid de tarjetas (alpha): Choya Piñata, Regente Chak, Furienhiedra, Shatterer, Tequatl, Dragonstorm, Triple Trouble.
- Cada tarjeta: imagen (marca de agua), descripción, **chatcode** con botón Copiar, icono de **infusión** como posible drop, y **toggle** "Hecho hoy" (persistencia por fecha).
- Wallet y Conversor permanecen funcionales y sin cambios de diseño.

Sugerencia de flujo Git: publicar en rama `feature/metaeventos-alpha` y luego abrir PR contra `main`.
