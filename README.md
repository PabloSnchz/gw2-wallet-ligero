## 🐾 ¿Qué es Bóveda del Gato Negro?

**Bóveda del Gato Negro** es una herramienta ligera y moderna para visualizar todas las divisas de tu cuenta de **Guild Wars 2**, con un estilo inspirado en ArenaNet y optimizaciones que mejoran la experiencia respecto al wallet tradicional.

Incluye:

- 🎨 **UI ArenaNet Style** con overlay y textura hero  
- 🔑 **Múltiples API Keys** guardadas localmente  
- 🗂️ **Tarjetas detalladas** con iconos, categorías y descripciones  
- 📊 **Vista compacta (tabla)** para comparar rápidamente  
- ⭐ **Favoritas / Bookmarks**  
- 🔍 **Filtros avanzados** (categoría, principales, solo >0, orden, búsqueda)  
- 🖼️ **Skeleton loading**  
- 🧩 **Modo modular** preparado para futuras features (MetaEventos, Conversor, etc.)
